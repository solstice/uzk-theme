UzK - ein Beamer-Theme für die Universität zu Köln

Allgemeines

UzK ist ein (inoffizielles) Theme für die latex-beamer-Klasse, das entsprechend den Vorgaben des Corporate Designs der Universität zu Köln entwickelt wurde.

Laden Sie hier eine Beispielpräsentation herunter.

Autor

David Kusterer

Verwendung

Mit dem folgenden LaTeX-Code erstellen Sie eine Beamer-Präsentation im Corporate Design der Universität zu Köln: 
\documentclass{beamer}

% Gliederung
% \useoutertheme{infolines}
% \useoutertheme{split}

\usetheme[%
wiso,       %% Wiso-Fakultaet
%jura,       %% Rechtswissenschaftliche Fakultaet
%medizin,    %% Medizinische Fakultaet
%philo,      %% Philosophische Fakultaet
%matnat,     %% Mathematisch-Naturwissenschaftliche Fakultaet
%human,      %% Humanwissenschaftliche Fakultaet
%verw,       %% Universitaetsverwaltung
%nav,        %% Schalte Navigationssymbole ein
%latexfonts, %% Verwende CM-Schriften
%colorful,   %% Verwende Fakultaetsfarben als primary, sec. etc. palette, ansonsten grau
]{UzK}

\title{Meine Präsentation}
\author{Hans Mustermann}

\begin{document}

\begin{frame}
\maketitle
\end{frame}

\end{document}

Dem Theme können verschiedene Optionen übergeben werden. Sie können aus verschiedenen Fußzeilen wählen: Standardmäßig wird der Balken in der Fußzeile in den Farben aller sechs Fakultäten sowie der Farbe der Verwaltung dargestellt. Möchten Sie den Balken jedoch einfarbig mit Ihrer jeweiligen Fakultätsfarbe gefüllt, so erreichen Sie dies mit den Optionen wiso, jura, medizin, philo, matnat, human und verw.

Für englische Präsentationen verwenden Sie die Option english des Pakets babel. Der Name der Universität sowie die Fakultätsnamen werden dann entsprechend angepasst.

Durch das Laden der outer themes infolines bzw. split wird die Gliederung der Präsentation im Seitenkopf angezeigt. Die Hintergrundfarben der Gliederung sind in Grautönen gehalten. Mit der Option colorful werden die jeweiligen Fakultätsfarben als Hintergrundfarben verwendet.

Aufgrund des Urheberrechts können wir die Logo-Dateien nicht öffentlich bereitstellen. Sie erhalten diese auf Anfrage bei den Ansprechpartnern für das Corporate Design.

Entwicklung

Wir freuen uns über jegliche Anmerkungen, Kritik etc. Falls Sie sich an der Entwicklung beteiligen möchten, können Sie das Theme einfach mit git klonen: 

$ git clone git://github.com/solstice/uzk-theme.git
Sie können den Quelltext ebenfalls als .zip oder .tar-Datei herunterladen.
